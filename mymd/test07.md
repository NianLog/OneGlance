## 数据的表示和运算

### 数制转换

```
十进制 → 二进制：除2取余，逆序排列
二进制 → 十进制：按权展开求和
十六进制：每4位二进制对应1位十六进制
```

<span class="tip-box">二进制是计算机内部唯一的数据表示方式。</span>

### 原码、反码、补码

| 编码 | 正数 | 负数 |
|------|------|------|
| 原码 | 符号位0，数值位正常表示 | 符号位1，数值位正常表示 |
| 反码 | 符号位0，数值位正常表示 | 符号位1，数值位取反 |
| 补码 | 符号位0，数值位正常表示 | 符号位1，数值位取反末位+1 |

### 定点数与浮点数

```c
// 定点数表示
int a = 100;  // 整数

// 浮点数表示（IEEE 754标准）
float b = 3.14;  // 32位：1位符号+8位指数+23位尾数
double c = 3.14159;  // 64位：1位符号+11位指数+52位尾数
```

<span class="warning-box">注意：浮点数存在精度问题，不能用于精确计算（如金融计算）。</span>

## 存储系统

### 存储器的分类

| 分类标准 | 类型 |
|---------|------|
| 存取方式 | 随机存取存储器（RAM）、顺序存取存储器（SAM） |
| 易失性 | 易失性存储器（内存）、非易失性存储器（硬盘） |
| 读写类型 | 只读存储器（ROM）、读写存储器（RAM） |

### 层次结构

```
┌─────────────────────────────────┐
│          CPU寄存器              │  速度最快，容量最小
├─────────────────────────────────┤
│         Cache（高速缓存）        │
├─────────────────────────────────┤
│      主存储器（内存）            │
├─────────────────────────────────┤
│      辅助存储器（硬盘）          │  速度最慢，容量最大
└─────────────────────────────────┘
```

<span class="tip-box">存储层次的设计目标：用较低的成本获得较高的平均访问速度。</span>

### 主存与CPU连接

```
CPU ←→ 地址总线 ←→ MAR（地址寄存器）
CPU ←→ 数据总线 ←→ MDR（数据寄存器）
CPU ←→ 控制总线 ←→ 控制单元
      ↓
    主存储器
```

## 指令系统

### 指令格式

```
┌─────┬───────┬───────────────┐
│ 操作 │ 地址码 │     操作数    │
│ 码   │       │               │
└─────┴───────┴───────────────┘
```

### 寻址方式

| 寻址方式 | 说明 | 示例 |
|---------|------|------|
| 立即寻址 | 操作数在指令中 | ADD #5 |
| 直接寻址 | 地址码给出内存地址 | ADD [100] |
| 间接寻址 | 地址码给出地址，地址中还有地址 | ADD @[100] |
| 寄存器寻址 | 操作数在寄存器中 | ADD R1 |
| 相对寻址 | PC相对偏移 | JMP +10 |

### 指令类型

1. **数据传送指令**：MOV、PUSH、POP
2. **算术运算指令**：ADD、SUB、MUL、DIV
3. **逻辑运算指令**：AND、OR、NOT、XOR
4. **控制转移指令**：JMP、CALL、RET
5. **输入输出指令**：IN、OUT

## 中央处理器

### CPU的基本结构

```
┌─────────────────────────────────────┐
│              CPU                    │
├──────────────────┬──────────────────┤
│     运算器        │      控制器      │
├──────────────────┼──────────────────┤
│ ALU │ 累加器 │   │ IR │ PC │ ID | CU│
│     │ 暂存器 │   │    │    │    │   │
└──────────────────┴──────────────────┘
```

### 指令周期

```
取指周期 → 执行周期 → 中断周期
    ↓          ↓          ↓
 FETCH    EXECUTE    INTERRUPT
```

### 流水线技术

```
时钟：  1   2   3   4   5   6   7
指令1： IF  ID  EX  MEM WB  -   -
指令2： -   IF  ID  EX  MEM WB  -
指令3： -   -   IF  ID  EX  MEM WB
```

**流水线加速比** ≈ 流水线级数

<span class="tip-box">流水线的关键：指令间无数据相关，或能通过流水线互锁解决。</span>

## 总线

### 总线分类

| 分类 | 说明 | 示例 |
|------|------|------|
| 片内总线 | CPU内部连接 | ALU内部总线 |
| 系统总线 | 计算机各部件连接 | 地址总线、数据总线、控制总线 |
| 通信总线 | 计算机间通信 | USB、PCI |

### 总线仲裁

解决多个设备同时请求使用总线的问题：

1. **集中仲裁方式**：链式查询、计数器定时查询、独立请求
2. **分布仲裁方式**：自举分布式、冲突检测分布式

### 总线标准

- ISA总线
- PCI总线
- AGP总线
- USB总线
- IEEE 1394总线
