# 操作系统复习指南

<span class="important-note">本指南涵盖操作系统核心知识点，包括进程管理、内存管理、文件系统等，适合备考和面试复习使用。</span>

## 第一章：进程管理

### 1.1 进程与线程

**进程**是操作系统资源分配的基本单位，**线程**是CPU调度的基本单位。

<span class="tip-box">关键点：进程拥有独立的地址空间，而线程共享进程的资源。</span>

#### 进程状态转换

```
就绪 → 运行 → 阻塞
 ↑     ↓      |
 └─────┴──────┘
```

### 1.2 进程调度算法

1. **先来先服务（FCFS）** - 简单但可能导致饥饿
2. **短作业优先（SJF）** - 平均等待时间最短
3. **时间片轮转（RR）** - 公平共享CPU
4. **多级反馈队列** - 综合性能最优

<span class="warning-box">注意：时间片轮转算法中，时间片过大退化为FCFS，过小则增加上下文切换开销。</span>

## 第二章：内存管理

### 2.1 分页与分段

- **分页**：将内存分成固定大小的页框
- **分段**：将程序分成逻辑段

### 2.2 虚拟内存

虚拟内存利用**局部性原理**，通过**请求调页**和**页面置换**实现。

<span class="tip-box">页面置换算法：FIFO、LRU（最近最少使用）、LFU（最不经常使用）</span>
